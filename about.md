# Find and Replace

Adds a Find and Replace button to the editor tab.

To use this mod, simply select objects you want to replace (or select all of them), then press Ctrl+H on desktop or press the Find and Replace button. Simple type the appropriate object IDs in the boxes and press Replace.

For a reference of all object IDs, check out Colon's level Object IDs New (99784974).
