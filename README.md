# Find and Replace

<img alt="find and replace logox" src="logo.png" width="150">

Adds a Find and Replace button to the editor tab.

To use this mod, simply select objects you want to replace (or select all of them), then press Ctrl+H on desktop or press the Find and Replace button. Simple type the appropriate object IDs in the boxes and press Replace.

Currently copies movement, rotation, scaling, and group IDs.

If there are any problems with this mod, fix it yourself and create a pull request (thanks!), or report it to the discord server


## Download

Download the mod from the [Actions tab](https://github.com/DarthStorm/find-and-replace/actions) or from [Releases](https://github.com/DarthStorm/find-and-replace/releases/)
